## Features

- **Kanban Board** for visual task management
- **Documentation Editor** with Markdown support
- **CI Monitor** for real-time build status
- **Dark Mode** with modern, responsive design

## Usage

This package is bundled with `@locusai/cli`. You typically don't need to install it directly.

```bash
# Start via CLI (recommended)
npx @locusai/cli dev

# Dashboard opens at http://localhost:3080
```
